# Student Feedback Evidence

This folder contains public-safe aggregate evidence from the classroom feedback survey used in the paper.

The original Microsoft Forms workbook contained student names and school email addresses. It has been removed from the public repository and replaced with aggregate summaries only.

## Response Set

- Total responses: 19
- Grade 10: 15 responses
- Grade 12: 4 responses

## Aggregate Category Means

| Category | Response count across items | Mean score |
| --- | ---: | ---: |
| Conceptual understanding | 57 | 4.21 |
| Engagement | 56 | 4.30 |
| Usability | 57 | 4.30 |
| Experimental skills | 57 | 4.28 |

## Files

- `student-feedback-aggregate.csv` gives item-level response counts and means on a five-point scale.
- `student-survey.docx` and `transformer-survey.docx` contain aggregate poll screenshots only.

No direct student identifiers are included in the public evidence files.
